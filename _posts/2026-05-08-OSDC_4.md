## Recon
First, I try to scan with nmap common port while waiting for scanning all port
`nmap -F <ip>`
```
PORT   STATE SERVICE
22/tcp open  ssh
80/tcp open  http
```

I will take a look at port `80` - the web of machine
I try fuzzing files on website using `ffuf`
`ffuf -u 'http://<ip>/FUZZ' -w /usr/share/wordlists/dirb/common.txt -t 100
`
```
css                     [Status: 301, Size: 170, Words: 5, Lines: 8, Duration: 53ms]

## Exploit
The description say that you need to brute force so I try username `admin` with some common password and found `happy`
After login successfully, It have a panel and a options to choose some command to the system, I try using burp suite proxy and then found it send via POST method with data like
```
radio=ls+-l&submit=Run
```
and then I try change radio value to `id` and it successfully have and os command injection.
Then I found it has `python3` so I go to revshells.com and create a python reverse shell and gain a shell on my machine.
Next I found that I'm currently running as `www-data` - a web account that do not have many permisson. Try to read `/etc/password`, then I found 3 normal users.

```
